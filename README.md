# Pulsifi Backend Side

Deploy with [Heroku](https://www.heroku.com). API URL [https://nestjs-pulsifi.herokuapp.com](https://nestjs-pulsifi.herokuapp.com).
* NestJS
* PostgreSQL
* TypeScript 
* JavaScript (ES6, ES7, ES8)
* JWT
* Passport

## Installation

```bash
$ npm install
```

## Running the app

```bash
# watch mode
$ npm run start:dev

# production mode
$ npm run build
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
